  1. ★★ State and prove the Monotone Convergence Theorem, or Beppo Levi Theorem. Q1.44
    Answer

    Proof sketch / Polya view: One inequality is monotonicity of the integral. For the reverse one, approximate the limit from below by a simple function, use the sets where \(f_n\) eventually dominates it, and then let the approximation parameters go to the end. Question focus: State and prove the Monotone Convergence Theorem, or Beppo Levi Theorem.

    Let \((f_n)\) be a sequence of non-negative measurable functions such that \(f_n \le f_{n+1}\) pointwise a.e., and \(f_n \to f\) pointwise a.e. The Monotone Convergence Theorem states:

    \[ \int_X f \, d\mu = \lim_{n\to\infty} \int_X f_n \, d\mu. \]
    Proof: Since \(f_n \le f\) a.e., monotonicity of the integral implies \(\int_X f_n \, d\mu \le \int_X f \, d\mu\). Taking the limit gives:
    \[ \lim_{n\to\infty} \int_X f_n \, d\mu \le \int_X f \, d\mu. \]
    To prove the reverse inequality, let \(s\) be any non-negative simple function such that \(0 \le s \le f\) pointwise. Fix any \(\varepsilon \in (0, 1)\). For each \(n \ge 1\), define the set:
    \[ E_n = \{ x \in X : f_n(x) \ge (1-\varepsilon) s(x) \}. \]
    1. Each \(E_n\) is measurable since \(f_n\) and \(s\) are measurable. 2. Since \(f_n \le f_{n+1}\), the sets are nested: \(E_n \subset E_{n+1}\). 3. We show \(\bigcup_{n=1}^\infty E_n = X\) (up to a null set). - If \(f(x) = 0\), then \(s(x) = 0\), so \(f_n(x) \ge 0 = (1-\varepsilon)s(x)\) holds for all \(n\), meaning \(x \in E_1\). - If \(f(x) > 0\), then \((1-\varepsilon)s(x) < f(x)\). Since \(f_n(x) \to f(x)\), there exists \(N\) such that \(f_n(x) \ge (1-\varepsilon)s(x)\) for all \(n \ge N\), meaning \(x \in E_N\). Thus \(E_n \uparrow X\) as \(n\to\infty\). 4. For any \(n \ge 1\), we have \(\int_X f_n \, d\mu \ge \int_{E_n} f_n \, d\mu \ge (1-\varepsilon) \int_{E_n} s \, d\mu\). 5. Write the simple function \(s\) as \(s = \sum_{j=1}^m a_j \chi_{A_j}\), where \(a_j \ge 0\) and \(A_j\) are disjoint measurable sets. Define the measure \(\nu(E) = \int_E s \, d\mu = \sum_{j=1}^m a_j \mu(E \cap A_j)\). Since \(\nu\) is a measure and \(E_n \uparrow X\), continuity of measure from below implies:
    \[ \lim_{n\to\infty} \int_{E_n} s \, d\mu = \lim_{n\to\infty} \nu(E_n) = \nu(X) = \int_X s \, d\mu. \]
    6. Taking the limit inferior as \(n \to \infty\) on both sides of the inequality in Step 4 yields:
    \[ \lim_{n\to\infty} \int_X f_n \, d\mu \ge (1-\varepsilon) \int_X s \, d\mu. \]
    7. Since this holds for any \(\varepsilon \in (0, 1)\), taking \(\varepsilon \downarrow 0\) yields \(\lim_{n\to\infty} \int_X f_n \, d\mu \ge \int_X s \, d\mu\). 8. Taking the supremum over all non-negative simple functions \(s \le f\) yields \(\lim_{n\to\infty} \int_X f_n \, d\mu \ge \int_X f \, d\mu\). Combining the two inequalities completes the proof.

  2. ★★ State and prove Fatou's Lemma. Q1.45
    Answer

    Proof sketch / Polya view: Build the proof from below. Replace the sequence by the increasing envelopes \(g_k=\inf_{m\ge k} f_m\), apply Monotone Convergence to \(g_k\), and compare each \(g_k\) with the tail integrals. Question focus: State and prove Fatou's Lemma.

    Let \((f_n)\) be a sequence of non-negative measurable functions. Fatou's Lemma states:

    \[ \int_X \liminf_{n\to\infty} f_n \, d\mu \le \liminf_{n\to\infty} \int_X f_n \, d\mu. \]
    Proof: 1. For each \(k \ge 1\), define the function \(g_k(x) = \inf_{m \ge k} f_m(x)\). 2. The sequence \((g_k)\) consists of non-negative measurable functions. Since the set of indices over which the infimum is taken shrinks as \(k\) increases, \(g_k(x) \le g_{k+1}(x)\) for all \(x \in X\). Thus, \((g_k)\) is pointwise monotonically increasing. 3. By definition, the pointwise limit is:
    \[ \lim_{k\to\infty} g_k(x) = \sup_{k\ge 1} \inf_{m \ge k} f_m(x) = \liminf_{n\to\infty} f_n(x). \]
    4. For any \(m \ge k\), we have \(g_k(x) \le f_m(x)\) for all \(x \in X\). By monotonicity of the integral:
    \[ \int_X g_k \, d\mu \le \int_X f_m \, d\mu \qquad \forall m \ge k. \]
    5. Taking the infimum over all \(m \ge k\) on the right-hand side yields:
    \[ \int_X g_k \, d\mu \le \inf_{m \ge k} \int_X f_m \, d\mu. \]
    6. Since \((g_k)\) is a monotonically increasing sequence of non-negative measurable functions converging pointwise to \(\liminf_{n\to\infty} f_n\), the Monotone Convergence Theorem implies:
    \[ \lim_{k\to\infty} \int_X g_k \, d\mu = \int_X \liminf_{n\to\infty} f_n \, d\mu. \]
    7. Taking the limit as \(k \to \infty\) on both sides of the inequality in Step 5:
    \[ \int_X \liminf_{n\to\infty} f_n \, d\mu = \lim_{k\to\infty} \int_X g_k \, d\mu \le \lim_{k\to\infty} \left( \inf_{m \ge k} \int_X f_m \, d\mu \right) = \liminf_{n\to\infty} \int_X f_n \, d\mu. \]

  3. ★★ State and prove the Lebesgue theorem. Q1.54
    Answer

    Proof sketch / Polya view: The goal is to justify passing the limit under the integral. State the domination hypothesis, prove the limit is still integrable, then use Fatou on a nonnegative dominated error to force the integral error to vanish. Question focus: State and prove the Lebesgue theorem.

    This is the Lebesgue Dominated Convergence Theorem.

    Let \((X,\mathcal A,\mu)\) be a measure space. Suppose \((f_n)\) is a sequence of measurable functions converging pointwise a.e. to \(f\), and there exists a non-negative integrable function \(g \in L^1(X, \mu)\) such that \(|f_n| \le g\) a.e. for every \(n \ge 1\). Then \(f \in L^1(X, \mu)\), \(f_n \in L^1(X, \mu)\) for all \(n\), and:

    \[ \lim_{n\to\infty} \int_X |f_n - f| \, d\mu = 0 \quad \text{and} \quad \lim_{n\to\infty} \int_X f_n \, d\mu = \int_X f \, d\mu. \]

    Proof: Since \(|f_n| \le g\) a.e. for all \(n\), passing to the limit pointwise a.e. yields \(|f| \le g\) a.e. Since \(g \in L^1(X, \mu)\), we have \(f \in L^1(X, \mu)\) and \(f_n \in L^1(X, \mu)\). We have \(|f_n - f| \le |f_n| + |f| \le 2g\) a.e., which implies \(2g - |f_n - f| \ge 0\) a.e. Applying Fatou's Lemma to the sequence of non-negative measurable functions \((2g - |f_n - f|)\):

    \[ \int_X \liminf_{n\to\infty} (2g - |f_n - f|) \, d\mu \le \liminf_{n\to\infty} \int_X (2g - |f_n - f|) \, d\mu. \]
    Since \(|f_n - f| \to 0\) a.e., the left-hand side is:
    \[ \int_X 2g \, d\mu. \]
    For the right-hand side, since \(g \in L^1\), we can pull the constant integral out of the limit inferior:
    \[ \liminf_{n\to\infty} \int_X (2g - |f_n - f|) \, d\mu = \int_X 2g \, d\mu - \limsup_{n\to\infty} \int_X |f_n - f| \, d\mu. \]
    Combining these yields:
    \[ \int_X 2g \, d\mu \le \int_X 2g \, d\mu - \limsup_{n\to\infty} \int_X |f_n - f| \, d\mu \implies \limsup_{n\to\infty} \int_X |f_n - f| \, d\mu \le 0. \]
    Since the integral of a non-negative function is non-negative, we get \(\lim_{n\to\infty} \int_X |f_n - f| \, d\mu = 0\). The convergence of the integrals follows because:
    \[ \left| \int_X f_n \, d\mu - \int_X f \, d\mu \right| \le \int_X |f_n - f| \, d\mu \to 0. \]

  4. ★★ In which case is it simple to find a dominating function? Q1.54
    Answer

    A dominating function \(g \in L^1(X, \mu)\) is typically found using one of the following mathematical strategies: 1. **Bounded Domains and Uniform Bounds:** If \(\mu(X) < \infty\) and there exists a constant \(M > 0\) such that \(|f_n(x)| \le M\) a.e. for all \(n \ge 1\), we choose \(g(x) = M\). Since \(\int_X M \, d\mu = M \mu(X) < \infty\), \(g\) is integrable. 2. **Pointwise Supremum / Decay Estimates:** If the domain is unbounded (e.g., \(\mathbb{R}^d\)), we seek an integrable function that bounds \(|f_n(x)|\) uniformly in \(n\). - For example, if \(|f_n(x)| \le \frac{C}{1 + |x|^{d+\alpha}}\) for some \(\alpha > 0\), then \(g(x) = \frac{C}{1 + |x|^{d+\alpha}}\) is integrable on \(\mathbb{R}^d\). - If there is an integrable singularity, such as \(|f_n(x)| \le \frac{C}{|x|^\alpha}\) for \(\alpha < d\) on a bounded neighborhood of \(0\), we choose \(g(x) = \frac{C}{|x|^\alpha}\). 3. **Monotonicity:** If the sequence is bounded by a known integrable function \(|f_n(x)| \le h(x)\) a.e. for all \(n\), then \(g = h\) is the dominating function.

2. Modes Of Convergence

  1. ★★★ For a sequence of functions \(\{f_n\}\in M\), write the definition of pointwise convergence. Q1.58
    Answer

    A sequence of measurable functions \((f_n)\) converges pointwise to \(f\) on \(X\) if for every fixed \(x \in X\):

    \[ \lim_{n\to\infty} f_n(x) = f(x). \]
    Equivalently, for every \(x \in X\) and every \(\varepsilon > 0\), there exists an integer \(N = N(x, \varepsilon)\) such that \(|f_n(x) - f(x)| < \varepsilon\) for all \(n \ge N\). The threshold \(N\) may depend on the point \(x\).

  2. ★★★ For a sequence of functions \(\{f_n\}\in M\), write the definition of almost everywhere convergence. Q1.58
    Answer

    A sequence of measurable functions \((f_n)\) converges almost everywhere to \(f\) (denoted by \(f_n \to f\) a.e.) if there exists a measurable set \(N \in \mathcal{A}\) with \(\mu(N) = 0\) such that for all \(x \in X \setminus N\):

    \[ \lim_{n\to\infty} f_n(x) = f(x). \]
    This means that the sequence converges pointwise at every point outside of a negligible set.

  3. ★★★ For a sequence of functions \(\{f_n\}\in M\), write the definition of convergence in measure. Q1.58
    Answer

    A sequence of measurable functions \((f_n)\) converges in measure to \(f\) if for every \(\varepsilon > 0\):

    \[ \lim_{n\to\infty} \mu(\{ x \in X : |f_n(x) - f(x)| > \varepsilon \}) = 0. \]
    This means that the measure of the set where the function values deviate by more than \(\varepsilon\) tends to zero as \(n \to \infty\).

  4. ★★★ For a sequence of functions \(\{f_n\}\in M\), write the definition of convergence in \(L^1\). Q1.58
    Answer

    A sequence of functions \((f_n)\) in \(L^1(X, \mu)\) converges in \(L^1\) (or converges in norm) to \(f \in L^1(X, \mu)\) if:

    \[ \lim_{n\to\infty} \|f_n - f\|_{L^1} = \lim_{n\to\infty} \int_X |f_n - f| \, d\mu = 0. \]

  5. ★★★ Under which hypothesis on \(X\) does convergence a.e. imply convergence in measure? Q1.61
    Answer

    The hypothesis required is that the measure space is finite, i.e., \(\mu(X) < \infty\). - *Proof:* Suppose \(f_n \to f\) a.e. Let \(S \subset X\) be the set where \(f_n(x) \to f(x)\), so \(\mu(X \setminus S) = 0\). Fix any \(\varepsilon > 0\). For each \(n \ge 1\), define:

    \[ E_n = \bigcup_{k=n}^\infty \{ x \in X : |f_k(x) - f(x)| > \varepsilon \}. \]
    The sets \(E_n\) are nested and decreasing: \(E_{n+1} \subset E_n\). Their intersection is \(E = \bigcap_{n=1}^\infty E_n\). If \(x \in E\), then \(|f_k(x) - f(x)| > \varepsilon\) for infinitely many \(k\), which means \(f_n(x) \not\to f(x)\). Thus, \(E \subset X \setminus S\), which implies \(\mu(E) = 0\). Since \(\mu(X) < \infty\), we have \(\mu(E_1) \le \mu(X) < \infty\). Applying the continuity of measure from above:
    \[ \lim_{n\to\infty} \mu(E_n) = \mu(E) = 0. \]
    Since \(\{ x \in X : |f_n(x) - f(x)| > \varepsilon \} \subset E_n\), by monotonicity:
    \[ \mu(\{ x \in X : |f_n(x) - f(x)| > \varepsilon \}) \le \mu(E_n) \to 0 \quad \text{as } n \to \infty, \]
    proving convergence in measure.

    \[ f_n = \chi_{[n, n+1]}. \]
    For any fixed \(x \in \mathbb{R}\), choosing \(n > x\) implies \(x \notin [n, n+1]\), so \(f_n(x) = 0\) for all \(n > x\). Thus, \(f_n \to 0\) pointwise everywhere on \(\mathbb{R}\).     \[ \lambda(\{ x \in \mathbb{R} : |f_n(x)| > \varepsilon \}) = \lambda([n, n+1]) = 1. \]
    Since this measure does not converge to \(0\) as \(n \to \infty\), the sequence does not converge to \(0\) in measure.

  7. ★★★ What is the relation between convergence in measure and convergence a.e. up to subsequences? Q1.60
    Answer

    If \(f_n \to f\) in measure, then there exists a subsequence \((f_{n_k})\) that converges to \(f\) almost everywhere. - **Subsequence Construction:** Since \(f_n \to f\) in measure, we can select a subsequence \((f_{n_k})\) such that for each \(k \ge 1\), the set \(A_k = \{ x \in X : |f_{n_k}(x) - f(x)| > 2^{-k} \}\) satisfies \(\mu(A_k) < 2^{-k}\). - **Borel-Cantelli Argument:** The sum of the measures of these sets is finite: \(\sum_{k=1}^\infty \mu(A_k) < \sum_{k=1}^\infty 2^{-k} = 1 < \infty\). By the First Borel-Cantelli Lemma, the set \(E = \limsup_{k\to\infty} A_k = \bigcap_{N=1}^\infty \bigcup_{k=N}^\infty A_k\) of points belonging to infinitely many \(A_k\) has measure \(\mu(E) = 0\). For any \(x \notin E\), there exists \(K(x)\) such that for all \(k \ge K(x)\), \(x \notin A_k\), which means \(|f_{n_k}(x) - f(x)| \le 2^{-k}\). Since \(2^{-k} \to 0\), \(\lim_{k\to\infty} f_{n_k}(x) = f(x)\) for all \(x \notin E\). Since \(\mu(E) = 0\), \(f_{n_k} \to f\) a.e.

  8. ★★★ Show that convergence in \(L^1\) implies convergence in measure. Q1.62
    Answer

    Proof sketch / Polya view: The wanted implication is an estimate between two notions of convergence. Fix a tolerance, convert a large-deviation set into an integral bound with Chebyshev/Markov, then send the \(L^1\) norm to zero. Question focus: Show that convergence in \(L^1\) implies convergence in measure.

    Assume \(f_n \to f\) in \(L^1(X, \mu)\). Fix any \(\varepsilon > 0\). By Chebyshev's inequality (or Markov's inequality), we have:

    \[ \mu(\{ x \in X : |f_n(x) - f(x)| > \varepsilon \}) \le \frac{1}{\varepsilon} \int_X |f_n - f| \, d\mu. \]
    Since \(f_n \to f\) in \(L^1\), the integral on the right-hand side tends to \(0\) as \(n \to \infty\). Since \(\varepsilon > 0\) is a fixed constant, we have:
    \[ \lim_{n\to\infty} \mu(\{ x \in X : |f_n(x) - f(x)| > \varepsilon \}) \le \frac{1}{\varepsilon} \cdot 0 = 0. \]
    Thus, \(f_n \to f\) in measure.

3. Lebesgue Points And Fundamental Theorems Of Calculus

  1. ★★★ Write the definition of Lebesgue point. Q1.66
    Answer

    Let \(f \in L^1_{\mathrm{loc}}(\mathbb{R}^n)\). A point \(x \in \mathbb{R}^n\) is a **Lebesgue point** of \(f\) if:

    \[ \lim_{r\downarrow0} \frac{1}{|B_r(x)|} \int_{B_r(x)} |f(y) - f(x)| \, dy = 0, \]
    where \(B_r(x)\) is the open ball of radius \(r\) centered at \(x\), and \(|B_r(x)|\) is its Lebesgue measure. This means the average oscillation of \(f\) around \(x\) vanishes as the ball shrinks to a point.

  2. ★★★ What is about the measure of the set of points that are not Lebesgue points for a function \(f\in L^1\)? Q1.66
    Answer

    By the Lebesgue Differentiation Theorem, if \(f \in L^1_{\mathrm{loc}}(\mathbb{R}^n)\), then almost every point \(x \in \mathbb{R}^n\) is a Lebesgue point of \(f\). Consequently, the set of points that are not Lebesgue points of \(f\) has Lebesgue measure zero.

    Let \(f \in L^1(a, b)\) and define \(F(x) = \int_a^x f(t) \, dt\) for \(x \in [a, b]\). Then \(F\) is absolutely continuous on \([a, b]\) and differentiable almost everywhere on \((a, b)\), with:

    \[ F'(x) = f(x) \qquad \text{for a.e. } x \in (a, b). \]
    Proof: Absolute continuity of \(F\) follows from the absolute continuity of the Lebesgue integral: for any finite family of disjoint intervals \((a_i, b_i) \subset [a, b]\), we have \(\sum_i |F(b_i) - F(a_i)| \le \int_{\bigcup_i (a_i, b_i)} |f(t)| \, dt\). Since \(f \in L^1\), for any \(\varepsilon > 0\), there exists \(\delta > 0\) such that \(\mu(E) < \delta \implies \int_E |f| < \varepsilon\). Choosing \(\sum_i (b_i - a_i) < \delta\) guarantees the sum of differences is less than \(\varepsilon\), so \(F\) is absolutely continuous. To show differentiability, let \(x \in (a, b)\) be a Lebesgue point of \(f\). We examine the difference quotient for \(h \neq 0\):
    \[ \frac{F(x+h) - F(x)}{h} - f(x) = \frac{1}{h} \int_x^{x+h} (f(t) - f(x)) \, dt. \]
    - For \(h > 0\):
    \[ \left| \frac{F(x+h) - F(x)}{h} - f(x) \right| \le \frac{1}{h} \int_x^{x+h} |f(t) - f(x)| \, dt \le \frac{2}{|B_{2h}(x)|} \int_{B_h(x)} |f(t) - f(x)| \, dt. \]
    - For \(h < 0\): Let \(\eta = -h > 0\).
    \[ \left| \frac{F(x+h) - F(x)}{h} - f(x) \right| \le \frac{1}{\eta} \int_{x-\eta}^x |f(t) - f(x)| \, dt \le \frac{2}{|B_{2\eta}(x)|} \int_{B_\eta(x)} |f(t) - f(x)| \, dt. \]
    In both cases, since \(x\) is a Lebesgue point of \(f\), taking \(h \to 0\) forces the right-hand side to \(0\). Thus, \(F'(x)\) exists and equals \(f(x)\) at every Lebesgue point. Since almost every point in \((a, b)\) is a Lebesgue point, \(F'(x) = f(x)\) a.e.

  4. ★★★ State the Second Fundamental Theorem of Calculus. Q1.68
    Answer

    Let \(F : [a, b] \to \mathbb{R}\). The function \(F\) is absolutely continuous on \([a, b]\) if and only if \(F\) is differentiable almost everywhere on \((a, b)\), \(F' \in L^1(a, b)\), and:

    \[ F(x) = F(a) + \int_a^x F'(t) \, dt \qquad \forall x \in [a, b]. \]
    This establishes that absolute continuity is the precise condition under which a function can be reconstructed by integrating its pointwise derivative.

4. Absolutely Continuous Functions And Sobolev Spaces

  1. ★★★ Write the definition of absolutely continuous function. Q1.69
    Answer

    A function \(u : [a, b] \to \mathbb{R}\) is **absolutely continuous** if for every \(\varepsilon > 0\), there exists \(\delta > 0\) such that for any finite family of pairwise disjoint subintervals \(\{(a_i, b_i)\}_{i=1}^N \subset [a, b]\):

    \[ \sum_{i=1}^N (b_i - a_i) < \delta \implies \sum_{i=1}^N |u(b_i) - u(a_i)| < \varepsilon. \]

  2. ★★★ Write the definition of weak derivative. Q1.70
    Answer

    Let \(u \in L^1_{\mathrm{loc}}(a, b)\). A function \(v \in L^1_{\mathrm{loc}}(a, b)\) is the **weak derivative** of \(u\) (written \(u' = v\)) if:

    \[ \int_a^b u(x) \varphi'(x) \, dx = - \int_a^b v(x) \varphi(x) \, dx \qquad \forall \varphi \in C_c^1(a, b). \]
    If such a function \(v\) exists, it is unique up to equality almost everywhere.

  3. ★★★ Write the definition of \(W^{1,p}\). Q1.70
    Answer

    For \(1 \le p \le \infty\), the Sobolev space \(W^{1, p}(a, b)\) is defined as:

    \[ W^{1, p}(a, b) = \{ u \in L^p(a, b) : u \text{ has a weak derivative } u' \in L^p(a, b) \}. \]
    It is equipped with the norm:
    \[ \|u\|_{W^{1, p}} = \left( \|u\|_{L^p}^p + \|u'\|_{L^p}^p \right)^{1/p} \quad (1 \le p < \infty), \]
    and \(\|u\|_{W^{1, \infty}} = \max \{ \|u\|_{L^\infty}, \, \|u'\|_{L^\infty} \}\) for \(p = \infty\).

  4. ★★★ State and prove the characterization of absolutely continuous functions in terms of the integration by parts formula. Q1.69
    Answer

    Proof sketch / Polya view: The problem is about translating between derivatives, integrals, and absolute continuity. Prove the easy direction by integrating the derivative, and the reverse direction by identifying the distributional derivative through test functions or integration by parts. Question focus: State and prove the characterization of absolutely continuous functions in terms of the integration by parts formula.

    For \(u \in L^1(a, b)\), \(u\) has an absolutely continuous representative on \([a, b]\) if and only if there exists \(v \in L^1(a, b)\) such that:

    \[ \int_a^b u(x) \varphi'(x) \, dx = - \int_a^b v(x) \varphi(x) \, dx \qquad \forall \varphi \in C_c^1(a, b). \]
    In this case, \(v\) is the weak derivative of \(u\).

    Proof: - **Necessity:** Suppose \(u\) is absolutely continuous. By the Second FTOC, \(u'\) exists a.e. and \(u' = v \in L^1(a, b)\). Integrating by parts for AC functions against \(\varphi \in C_c^1(a, b)\) yields:

    \[ \int_a^b u(x) \varphi'(x) \, dx = [u(x)\varphi(x)]_a^b - \int_a^b u'(x) \varphi(x) \, dx = - \int_a^b v(x) \varphi(x) \, dx, \]
    since \(\varphi(a) = \varphi(b) = 0\). - **Sufficiency:** Assume there exists \(v \in L^1(a, b)\) satisfying the relation. Define the absolutely continuous function \(w(x) = \int_a^x v(t) \, dt\). By the necessity part, \(w\) satisfies \(\int_a^b w \varphi' \, dx = - \int_a^b v \varphi \, dx\). Subtracting the two equations gives:
    \[ \int_a^b (u(x) - w(x)) \varphi'(x) \, dx = 0 \qquad \forall \varphi \in C_c^1(a, b). \]
    We use the **Lemma:** If \(h \in L^1(a, b)\) satisfies \(\int_a^b h \varphi' \, dx = 0\) for all \(\varphi \in C_c^1(a, b)\), then \(h\) is constant a.e. - *Proof of Lemma:* Fix \(\psi \in C_c^\infty(a, b)\) with \(\int_a^b \psi \, dx = 1\). For any \(f \in C_c^\infty(a, b)\), let \(C_f = \int_a^b f \, dx\). The function \(\eta = f - C_f \psi \in C_c^\infty(a, b)\) integrates to 0. Define \(\varphi(x) = \int_a^x \eta(t) \, dt \in C_c^1(a, b)\). Since \(\varphi' = f - C_f \psi\):
    \[ 0 = \int_a^b h \varphi' \, dx = \int_a^b h(x) (f(x) - C_f \psi(x)) \, dx = \int_a^b \left[ h(x) - \left(\int_a^b h(t)\psi(t)\,dt\right) \right] f(x) \, dx. \]
    Since this holds for all \(f \in C_c^\infty(a, b)\), we have \(h(x) = \int_a^b h(t)\psi(t)\,dt = \text{constant}\) a.e. Applying this Lemma to \(h = u - w\), we get \(u(x) - w(x) = C\) a.e., meaning \(u(x) = C + \int_a^x v(t) \, dt\) a.e., which is absolutely continuous.

  5. ★★★ State another characterization of absolutely continuous functions in terms of \(W^{1,1}\) and AC. Q1.71
    Answer
    Conceptual Bridge. The classical definition of absolute continuity is an \(\varepsilon\)-\(\delta\) condition on finite collections of disjoint intervals: small total interval length forces small total oscillation. The characterization below turns that external control condition into a structural description:
    \[ u \in AC([a,b]) \iff u(x)=u(a)+\int_a^x g(t)\,dt \quad \text{for some } g\in L^1(a,b). \]
    The key takeaway is: absolutely continuous functions are exactly the indefinite integrals of \(L^1\) functions. The same function \(g\) is simultaneously the classical derivative of \(u\) almost everywhere and the weak derivative of \(u\).

    A function \(u : [a, b] \to \mathbb{R}\) is absolutely continuous if and only if there exists \(g \in L^1(a, b)\) such that:

    \[ u(x) = u(a) + \int_a^x g(t) \, dt \qquad \forall x \in [a, b]. \]
    In this case, \(g = u'\) almost everywhere in the classical sense, and the same \(g\) is the weak derivative of \(u\). Equivalently, every equivalence class in \(W^{1,1}(a,b)\) contains a unique absolutely continuous representative, obtained by integrating its weak derivative.

  6. ★★★ What is the relation between \(W^{1,1}\) and AC? Justify the answer. Q1.71
    Answer

    Proof sketch / Polya view: The problem is about translating between derivatives, integrals, and absolute continuity.     \[ \tilde u(x) = \tilde u(a) + \int_a^x u'(t)\,dt. \]
    This reconstructed curve is absolutely continuous and is the canonical representative of the Sobolev equivalence class.

    1. If \(u \in W^{1,1}(a,b)\), its weak derivative \(u'\) is in \(L^1(a,b)\). By the characterization theorem, the Sobolev class of \(u\) has the absolutely continuous representative

5. Measurability And Characteristic-Function Tests

  1. ★★★ Write the definition of measurable function. State the theorem about the measurability of the composite function. Q1.20, Q1.25
    Answer

    Let \((X, \mathcal{A})\) and \((Y, \mathcal{B})\) be measurable spaces. A function \(f : X \to Y\) is measurable if:

    \[ f^{-1}(B) \in \mathcal{A} \qquad \forall B \in \mathcal{B}. \]
    - **Theorem (Composition):** If \(f : (X, \mathcal{A}) \to (Y, \mathcal{B})\) is measurable and \(g : (Y, \mathcal{B}) \to (Z, \mathcal{C})\) is measurable, then their composition \(g \circ f : (X, \mathcal{A}) \to (Z, \mathcal{C})\) is measurable. *Proof:* For any \(C \in \mathcal{C}\), we have \((g \circ f)^{-1}(C) = f^{-1}(g^{-1}(C))\). Since \(g\) is measurable, \(g^{-1}(C) \in \mathcal{B}\). Since \(f\) is measurable, \(f^{-1}(g^{-1}(C)) \in \mathcal{A}\). - **Critical Warning:** If \(f : \mathbb{R} \to \mathbb{R}\) and \(g : \mathbb{R} \to \mathbb{R}\) are Lebesgue measurable, their composition \(g \circ f\) is **not necessarily** Lebesgue measurable. This is because \(g^{-1}(C)\) for a Borel set \(C\) is Lebesgue measurable but not necessarily Borel, and \(f\) is only guaranteed to pull back Borel sets to Lebesgue measurable sets. However, if \(g\) is **Borel measurable** and \(f\) is Lebesgue measurable, then \(g \circ f\) is Lebesgue measurable.

6. Low But Not Impossible Surprise Candidates

  1. ★ State the Caratheodory measurability condition and the theorem that turns an outer measure into a complete measure. Q1.10, Q1.11, Q1.12, Q1.13
    Answer

    Let \(\mu^*\) be an outer measure on \(X\). A set \(E\subset X\) is Caratheodory measurable if for every \(A\subset X\),

    \[ \mu^*(A)=\mu^*(A\cap E)+\mu^*(A\setminus E). \]
    Since subadditivity always gives \(\le\), it is enough to prove the reverse inequality. Every set of outer measure zero is Caratheodory measurable. The Caratheodory theorem states that the family of all \(\mu^*\)-measurable sets is a \(\sigma\)-algebra and the restriction of \(\mu^*\) to it is a complete measure.

  2. ★ Explain the construction of Lebesgue measure on \(\mathbb R\) and on \(\mathbb R^N\). Q1.14
    Answer

    Start from the Lebesgue outer measure. In \(\mathbb R\), cover \(E\subset\mathbb R\) by countable families of intervals and set

    \[ \lambda^*(E)=\inf\left\{\sum_{k=1}^\infty |I_k|: E\subset\bigcup_{k=1}^\infty I_k\right\}. \]
    In \(\mathbb R^N\), use rectangles or boxes and replace \(|I_k|\) by their \(N\)-dimensional volume. Lebesgue measurable sets are the Caratheodory measurable sets for this outer measure, and Lebesgue measure is the restriction of \(\lambda^*\) to that \(\sigma\)-algebra.

  3. ★ Write the definition of integrable functions, the Lebesgue integral, and \(L^1\). Q1.49
    Answer

    For a measurable real-valued function \(f\), write \(f=f^+-f^-\), where \(f^+=\max\{f,0\}\) and \(f^-=\max\{-f,0\}\). The function \(f\) is integrable if both \(\int f^+\,d\mu\) and \(\int f^-\,d\mu\) are finite. Then

    \[ \int_X f\,d\mu=\int_X f^+\,d\mu-\int_X f^-\,d\mu. \]
    The space \(L^1(X,\mu)\) is the vector space of measurable functions with finite integral of the absolute value, modulo equality a.e.:
    \[ L^1(X,\mu)=\{f:\int_X |f|\,d\mu<\infty\}/\sim. \]

  4. ★ Given a metric space \((X,d)\), write the definition of its Borel \(\sigma\)-algebra \(\mathcal B(X)\). Provide classes of Borel sets and characterize \(\mathcal B(\mathbb R)\), \(\mathcal B(\mathbb R^N)\). Q1.3
    Answer

    For a metric space \((X, d)\), the Borel \(\sigma\)-algebra \(\mathcal{B}(X)\) is defined as the smallest \(\sigma\)-algebra containing all open subsets of \(X\):

    \[ \mathcal{B}(X) = \sigma(\{ U \subset X : U \text{ is open} \}). \]
    Classes of Borel sets include: 1. **Open Sets** and **Closed Sets**. 2. **\(F_\sigma\) sets:** Countable unions of closed sets. 3. **\(G_\delta\) sets:** Countable intersections of open sets. 4. **\(F_{\sigma\delta}\) sets** (countable intersections of \(F_\sigma\) sets) and **\(G_{\delta\sigma}\) sets** (countable unions of \(G_\delta\) sets). **Characterization on \(\mathbb{R}\):** \(\mathcal{B}(\mathbb{R})\) can be generated by any of the following families of intervals (with endpoints in \(\mathbb{R}\) or \(\mathbb{Q}\)): - Open intervals: \(\{(a, b) : a, b \in \mathbb{R}\}\). - Closed intervals: \(\{[a, b] : a, b \in \mathbb{R}\}\). - Half-open intervals: \(\{(a, b] : a, b \in \mathbb{R}\}\) or \(\{[a, b) : a, b \in \mathbb{R}\}\). - Unbounded open intervals: \(\{(-\infty, a) : a \in \mathbb{R}\}\) or \(\{(a, \infty) : a \in \mathbb{R}\}\). **Characterization on \(\mathbb{R}^N\):** \(\mathcal{B}(\mathbb{R}^N)\) is generated by open balls, by open rectangles of the form \(\prod_{j=1}^N (a_j, b_j)\), or by rational rectangles.

  5. ★ Prove the continuity of the measure along monotone sequences, either increasing or decreasing, under suitable further assumptions. Q1.5
    Answer

    Proof sketch / Polya view: The task asks for the two monotone-set limit rules. For increasing sets, decompose into disjoint increments; for decreasing sets, reduce to the increasing case by complements and use the finite-measure hypothesis. Question focus: Prove the continuity of the measure along monotone sequences, either increasing or decreasing, under suitable further assumptions.

    Let \((X, \mathcal{A}, \mu)\) be a measure space. 1. **Increasing Sequences:** If \(E_n \uparrow E = \bigcup_{n=1}^\infty E_n\), then \(\lim_{n\to\infty} \mu(E_n) = \mu(E)\). - *Proof:* Define \(F_1 = E_1\), and \(F_n = E_n \setminus E_{n-1}\) for all \(n \ge 2\). The sets \((F_n)\) are pairwise disjoint and satisfy \(\bigcup_{n=1}^N F_n = E_N\) and \(\bigcup_{n=1}^\infty F_n = E\). By countable additivity:

    \[ \mu(E) = \sum_{n=1}^\infty \mu(F_n) = \lim_{N\to\infty} \sum_{n=1}^N \mu(F_n) = \lim_{N\to\infty} \mu(E_N). \]
    2. **Decreasing Sequences:** If \(E_n \downarrow E = \bigcap_{n=1}^\infty E_n\) and \(\mu(E_1) < \infty\), then \(\lim_{n\to\infty} \mu(E_n) = \mu(E)\). - *Proof:* Define \(A_n = E_1 \setminus E_n\). Since \(E_n\) is decreasing, \(A_n\) is increasing: \(A_n \uparrow \bigcup (E_1 \setminus E_n) = E_1 \setminus E\). Applying the increasing case yields:
    \[ \lim_{n\to\infty} \mu(E_1 \setminus E_n) = \mu(E_1 \setminus E). \]
    Since \(\mu(E_1) < \infty\), we can write \(\lim_{n\to\infty} (\mu(E_1) - \mu(E_n)) = \mu(E_1) - \mu(E) \implies \lim_{n\to\infty} \mu(E_n) = \mu(E)\). - **Counterexample for decreasing sequences without finite measure:** Let \(X = \mathbb{R}\) with Lebesgue measure \(\lambda\). Let \(E_n = [n, \infty)\). We have \(E_n \downarrow \varnothing\), so \(\lambda(\bigcap E_n) = 0\). However, \(\lambda(E_n) = \infty\) for every \(n \ge 1\). Thus, \(\lim_{n\to\infty} \lambda(E_n) = \infty \neq 0\).

  6. ★ State and prove the theorem about regularity of the Lebesgue measure. Q1.18
    Answer

    Proof sketch / Polya view: The proof is an approximation argument. State inner and outer regularity, prove it first for convenient measurable sets, then extend by monotone approximation and the definition of Lebesgue measure. Question focus: State and prove the theorem about regularity of the Lebesgue measure.

    Lebesgue measure \(\lambda\) on \(\mathbb{R}^d\) is regular: for any Lebesgue measurable set \(E \subset \mathbb{R}^d\):

    \[ \lambda(E) = \inf \{ \lambda(O) : E \subset O, \, O \text{ open} \} \quad (\text{Outer Regularity}) \]
    \[ \lambda(E) = \sup \{ \lambda(K) : K \subset E, \, K \text{ compact} \} \quad (\text{Inner Regularity}) \]
    Proof: - **Outer Regularity:** By definition of the Lebesgue outer measure, \(\lambda(E) = \inf \sum_{j=1}^\infty |I_j|\), where the infimum is taken over all countable coverings of \(E\) by open boxes \(I_j\). For any \(\varepsilon > 0\), choose a covering of \(E\) by open boxes \(I_j\) such that \(\sum_{j=1}^\infty |I_j| < \lambda(E) + \varepsilon\). Let \(O = \bigcup_{j=1}^\infty I_j\). Then \(O\) is an open set containing \(E\), and by subadditivity:
    \[ \lambda(O) \le \sum_{j=1}^\infty \lambda(I_j) = \sum_{j=1}^\infty |I_j| < \lambda(E) + \varepsilon. \]
    Taking the infimum over all open sets \(O \supset E\) yields Outer Regularity. - **Inner Regularity:** - *Case 1: \(E\) is bounded.* Let \(E\) be contained in a bounded open box \(U \subset \mathbb{R}^d\). Its complement in \(U\), \(F = U \setminus E\), is also measurable. By Outer Regularity applied to \(F\), for any \(\varepsilon > 0\), there exists an open set \(O \supset F\) such that \(\lambda(O) < \lambda(F) + \varepsilon\). Let \(K = U \setminus O\). Since \(O\) is open, \(K\) is closed, and since \(K \subset U\), \(K\) is bounded, so \(K\) is compact. Since \(O \supset U \setminus E\), we have \(K = U \setminus O \subset E\). We have \(\lambda(U) = \lambda(O) + \lambda(K)\) and \(\lambda(U) = \lambda(E) + \lambda(F)\). Therefore:
    \[ \lambda(E) = \lambda(U) - \lambda(F) > \lambda(U) - (\lambda(O) - \varepsilon) = \lambda(K) + \varepsilon. \]
    Thus \(\lambda(K) > \lambda(E) - \varepsilon\), proving Inner Regularity for bounded sets. - *Case 2: \(E\) is unbounded.* For each \(R \ge 1\), define \(E_R = E \cap B_R(0)\). Since \(E_R \uparrow E\), continuity of measure from below implies \(\lambda(E_R) \to \lambda(E)\). For any \(L < \lambda(E)\), choose \(R\) large enough such that \(\lambda(E_R) > L\). Since \(E_R\) is bounded, Case 1 yields a compact set \(K \subset E_R \subset E\) such that \(\lambda(K) > L\), proving Inner Regularity generally.

  7. ★ State and prove the Vanishing Lemma in the following two cases: for nonnegative measurable functions and for functions belonging to \(L^1\). Q1.43, Q1.52
    Answer

    Proof sketch / Polya view: The proof wants measurability reduced to preimages or level sets. Rewrite the target function using countable unions/intersections and known measurable building blocks, then use closure of the sigma-algebra. Question focus: State and prove the Vanishing Lemma in the following two cases: for nonnegative measurable functions and for functions belonging to \(L^1\).

    Case 1 (Nonnegative functions): If \(f \in M^+(X, \mathcal{A})\) and \(\int_X f \, d\mu = 0\), then \(f = 0\) a.e.

    Proof: For each \(n \ge 1\), define the set \(E_n = \{ x \in X : f(x) \ge 1/n \}\). By Chebyshev's inequality:

    \[ 0 = \int_X f \, d\mu \ge \int_{E_n} f \, d\mu \ge \frac{1}{n} \mu(E_n) \implies \mu(E_n) = 0. \]
    The set where \(f\) is positive is \(\{ x \in X : f(x) > 0 \} = \bigcup_{n=1}^\infty E_n\). By countable subadditivity:
    \[ \mu(\{ f > 0 \}) \le \sum_{n=1}^\infty \mu(E_n) = 0. \]
    Thus \(f = 0\) a.e.

    Case 2 (L^1 functions): If \(f \in L^1(X, \mu)\) and \(\int_E f \, d\mu = 0\) for every measurable set \(E \in \mathcal{A}\), then \(f = 0\) a.e.

    Proof: Let \(E_+ = \{ x \in X : f(x) > 0 \}\) and \(E_- = \{ x \in X : f(x) < 0 \}\). Both sets are measurable. By hypothesis:

    \[ \int_{E_+} f \, d\mu = 0 \quad \text{and} \quad \int_{E_-} f \, d\mu = 0. \]
    Since \(f = f^+\) on \(E_+\) and \(f = -f^-\) on \(E_-\), we have:
    \[ \int_X f^+ \, d\mu = \int_{E_+} f \, d\mu = 0 \quad \text{and} \quad \int_X f^- \, d\mu = \int_{E_-} (-f) \, d\mu = 0. \]
    Applying Case 1 to the nonnegative functions \(f^+\) and \(f^-\) yields \(f^+ = 0\) a.e. and \(f^- = 0\) a.e. Thus \(f = f^+ - f^- = 0\) a.e.
